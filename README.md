# 🧠 OOP in Python Practice

This repository contains my practice codes for **Object-Oriented Programming (OOP) in Python**.
I have previously studied OOP concepts in **C++** and am now implementing them in Python to strengthen my understanding.

---

## 📌 Topics Covered

* Classes and Objects
* Class Attributes
* Instance Variables
* Methods
* Basic Input/Output in Classes

---

## 📂 Files

### 1. `student_class_basic.py`

* Demonstrates basic class creation
* Uses class attributes
* Shows how objects access class data

### 2. `employee_class_input_output.py`

* Demonstrates instance variables
* Uses methods inside a class
* Takes user input and displays output
